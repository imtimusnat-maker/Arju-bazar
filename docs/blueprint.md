# **App Name**: Your Shop

## Core Features:

- Product Catalog: Display a comprehensive catalog of products across various categories: Men, Women, Kids collections, 3D waterproof bedsheets, and furniture. Same to same function from https://ghorerbazar.com/
- Product Detail Pages: Detailed product pages with images, descriptions, and pricing.
- Shopping Cart: A shopping cart system to add, remove, and manage selected items.
- Checkout Process: A streamlined checkout process including shipping options and payment gateway integration.
- User Accounts: User registration and login functionality to manage orders and personal information.
- Admin Panel: An admin panel to manage products, categories, users, and orders. This will provide full control of the website, to manage the inventory, make special offers etc.

## Style Guidelines:

- Primary color: Orange (#FF8C00), matching the energetic and inviting feel of Ghorer Bazar.
- Background color: Light beige (#F5F5DC), offering a neutral and clean backdrop for the products.
- Accent color: Light orange (#FFA500), to draw the users' attention to key elements, calls to action and sale items.
- Body font: 'PT Sans' for readability and a modern touch.
- Headline font: 'Playfair' to convey the fashionable aspect of the items on sale. The body will still be 'PT Sans' though.
- Simple, outline-style icons for categories and UI elements to maintain a clean and modern aesthetic, mirroring the style of Ghorer Bazar.
- Grid-based layout with clear sections for different product categories, consistent with the reference website.
- Subtle animations for transitions and interactions to enhance user experience.
